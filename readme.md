# Canvas Particle Animation

A simple particle animation created using **HTML5 Canvas**, **CSS**, and **JavaScript**. The project demonstrates smooth particle movement and basic animation techniques using the Canvas API.

## Features

- Smooth particle animation
- Random particle movement
- Responsive canvas
- Lightweight and browser-based

## Technologies Used

- HTML5
- CSS3
- JavaScript
- HTML5 Canvas API

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Enjoy the animation!
